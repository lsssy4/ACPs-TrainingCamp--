# ACPs-TrainingCamp--
这段主程序代码实现了一个双通道数字锁相环（PLL）系统，用于锁定两个输入信号（A和B）的频率和相位，并同时控制DDS输出同频同相信号。
  程序通过FFT分析混合信号C，分离出A、B的频率和初始相位。优先检测出目标信号频率，控制输出信号频率向目标信号频率靠近，随后将输出信号继续与目标信号进行相位分析，计算二者相位差（对相位差进行归一化以避免相位跳跃问题），在锁相循环中，系统比较相邻两次的相位差，判断频率调整方向，用过高频相位差采样，对比调整信号频率（初始频率锁定过程中采用大步进，相位锁定过程中依据DDS-AD9959硬件性能采用精细频率步进），直至相位差趋于零，实现锁定。
